# Agora Calling App

A Flutter-based real-time calling application using Agora SDK and Firebase.

This project contains two separate Flutter applications:

- Caller App
- Receiver App

The applications support both audio and video calling with Firebase-based realtime communication management.

---

# Features

## Authentication
- Email & Password Login
- Email & Password Registration
- Persistent Login using SharedPreferences

## Calling Features
- Audio Calling
- Video Calling
- Incoming Call Screen
- Call Accept / Reject
- Realtime Call Status Updates
- Call Duration Timer
- Speaker Control
- Mute / Unmute
- Camera Toggle
- Camera Switch

## Additional Features
- Firebase Cloud Messaging (FCM)
- Firestore Realtime Sync
- Call History Screen
- Modern UI Design
- Provider State Management

---

# Tech Stack

- Flutter
- Provider
- Firebase Authentication
- Cloud Firestore
- Firebase Cloud Messaging
- Agora RTC Engine
